# From Fujian to Filipino Flavors
### A Comparative Study of Southern Chinese and Cebuano Cuisine Within Philippine Regional Food Traditions

> **Author:** Go, Thea (吴珊珊)

## 📚 About This Project

This repository hosts an interactive, web-based visualization of my academic paper examining the culinary influence of Southern China (Fujian Province) on Cebuano cuisine in the Philippines. 

Through qualitative analysis of four iconic dishes—*humba*, *ngohiong*, *tikoy*, and *maci*—this study traces their origins to specific Fujianese preparations and demonstrates how historical trade networks, Chinese settlement patterns, and geographic connectivity enabled deeper cultural transmission in Cebu than in other Philippine regions.

### 🔑 Key Contributions
- Challenges Manila-centric narratives of Chinese-Filipino heritage
- Documents Cebu as a unique node of Sino-Philippine cultural continuity
- Positions food as living cultural diplomacy in Philippines-China relations

## 🚀 How to Use

### Option 1: View Locally
1. Download or clone this repository
2. Open `index.html` directly in any modern web browser
3. All styles load via CDN—no installation required

### Option 2: Host on GitHub Pages
1. Upload all files to a new GitHub repository
2. Go to **Settings → Pages**
3. Select branch: `main` → Folder: `/ (root)`
4. Save → Your site will be live at:  
   `https://yourusername.github.io/your-repo-name/`

### Option 3: Quick Share via Netlify
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag the entire project folder onto the upload zone
3. Copy the generated link to share instantly
